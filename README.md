# Laboratorio 53 - Creación de módulo evaluación de desempeño de empleados

El siguiente proyecto tiene como objetivo desarrollar un nuevo módulo para Odoo que gestione las evaluaciones de desempeño de los empleados de una empresa.

Por lo tanto, cuando realicemos el ejercicio y facilitar su comprensión, vamos a adjuntar una serie de capturas de pantalla de la funcionalidad de la misma.
